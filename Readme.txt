Установка и запуск:
-- Ubuntu 16.04 LTS:
    sudo apt-get install libcgal-dev
    cd Mashgraph_Task4_RayTracing
    make
-- MacOS
    Установить XCode
    Установить HomeBrew
    brew install cgal
    Открыть проект Mashgraph_Task4_RayTracing.xcodeproj в XCode

В папке bin лежат 3 файла с разными настройками:
config_high_qual.txt: высокое качество (~6 часов при выполнении в 8-поточном режиме)
config_fast.txt: плохое качество (~2 минуты при выполнении в 8-поточном режиме)
config_other_LED_pos.txt: плохое качество + фонарик в другом месте пространства (~2 мин.)

Ко всем интуитивно не очень понятным пунктам в config-файлах имеются комментарии.

Реализованы:
-- База (10 баллов)
-- Освещение фонариком всех объектов в сцене (+5 баллов)
